📱 第一阶段：使用爱思助手进行签名
这个阶段的目标是把你手里的 WDA.ipa 文件，用你的 Apple ID 签上名。

1. 准备工作
一台 Windows 电脑和一根 数据线。
从官网下载并安装 最新版爱思助手。
一个 Apple ID。如果你担心安全问题，强烈建议专门注册一个不常用的“小号”Apple ID，不要使用主账号。
你之前获得的 WDA.ipa 文件。
2. 开始签名操作
连接设备与启动工具：用数据线把 iPhone 连上电脑并打开爱思助手。手机上如果弹出“要信任此电脑吗？”，点击“信任”并输入锁屏密码。
进入签名界面：在爱思助手主界面顶部点击 “工具箱”，然后在列表中找到并点击 “IPA 签名”。
添加 IPA 文件：在弹出的窗口中，点击 “添加 IPA 文件”，选择你准备好的 WDA.ipa 文件；或者直接把文件拖拽进窗口。
添加 Apple ID：在右侧的“使用 Apple ID 签名”区域，点击 “添加 Apple ID”，输入你的 Apple ID 和密码。
特别提醒：如果你的 Apple ID 开启了双重认证，这里不能直接填密码，而是需要登录 Apple ID 官网生成一个 “App 专用密码” ，然后把这个专用密码填进去。
执行签名：勾选你刚刚添加的 IPA 文件和 Apple ID，点击 “开始签名” 按钮。等待几秒钟，看到“签名成功”的提示就可以了。

📲 第二阶段：安装与授权
签名成功后，需要把生成的 IPA 安装到手机上，并进行最后的授权。
1. 安装到手机
签名成功后，爱思助手会自动弹出签名文件所在的文件夹。你可以直接在那个文件夹里找到已签名的 IPA 文件。
回到爱思助手主界面，点击 “应用游戏” -> “导入安装”，选择刚才签名好的 IPA 文件，它就会自动安装到你的手机上。

2. 在手机上信任证书（非常重要）
这是很多新手会遗漏的一步。如果不做，打开应用会闪退或提示“未受信任的开发者”。

安装完成后，先不要着急打开 App。

在 iPhone 上，打开 “设置” → “通用” → “VPN 与设备管理”（旧版 iOS 可能叫“描述文件与设备管理”）。

在这里你会看到你的 Apple ID 邮箱或者一个开发者名称，点击它。

然后点击 “信任...”，在弹窗中再次确认 “信任”。

完成这一步后，就可以回到桌面正常打开 WDA 了。

⚠️ 特别提醒：关于 7 天有效期
时效性：使用免费的 Apple ID 签名，应用的有效期只有 7 天。7 天后，WDA 会无法启动，需要你重复“删除手机上的旧应用 → 用爱思助手重新签名 → 重新安装” 这一整套流程。

数量限制：同一个 Apple ID 在 7 天内，最多只能给 3 个应用签名。
